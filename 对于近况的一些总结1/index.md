# 对于近况的一些总结（1）

这是本系列的第一篇文章，我希望通过这个系列来进行一个定期的**内省**，明确自己到底干了些什么，接下来往哪里走。

## 简述

基本上从大三一开学，就暂时放下了 ACM 的训练，主要原因是想**学点别的**。事实上，两个月也学了不少乱七八糟的东西：
- git
- vim
- docker
- gdb/lldb
- make/cmake
- hugo
- CS144
- 科研入门
以及一些新的工作流。

可以看到大多是工具，主要是之前都是需要用的时候稍微学一点点，久而久之就不满足了。只能说是补一点漏吧，有点 MIT *The Missing Semester of Your CS Education* 内味了。

随着视野的扩展，我当前对底层（主要指 EE）有了点兴趣，但感觉花时间在这上面不利于找工作，搞不好连饭都吃不了了。总之就处于这种啥都想学，但是怕走错方向吃不上饭的矛盾心理。

也许，是时候放下天真的想法了。

最近几天学校的作业还算比较密集，有几门比较棘手的学科，也有几个比较麻烦的大作业。所以我选择重构一下博客。（？）主要是博客的到期更加紧迫一点，而且我忍受 `Wordpress` 已经很久了。

## 计划

在早些时候，我的近期计划是 *把 CS144 写完，就再冲刺一下 ACM* 。但由于各种原因，CS144 的完成时间大大推迟了，甚至到现在还未完成。更确切地说，还没有到我觉得能够放过这门课的阶段。

因此，接下来我会全力攻克CS144，接下来就要补一下ACM了。然而，计组和操作系统还没有学也是非常紧迫的问题。

原本想在寒假找份实习，但看下来时间安排不过来，基本都是要求实习三个月以上的。但是我认为寒假在家里也不见得能学到多少东西，还是得想想办法的。（最理想的状态是在实习的时候还能把MIT6.824给写了）

## 内省

我对自己近期的效率是不甚满意的。虽然有很多借口可以找，但确实是没有找到一个合适的工作状态，脑子压根就没转起来。这可能和我的生活习惯导致的身体状态有关，也与我的工作习惯相关。我认为，是时候做出改变了。最最基本的就是早睡早起的规律作息，再加上一些日常的运动习惯。

最大的问题是，最近的学习已经很久没有成体系了。感觉比较确切的说法是，碎片化学习。个人感觉很不好，没有了知识的反复记忆，也没有输出的过程，无论是理解还是记忆上都不太理想。也是需要想想办法了。
